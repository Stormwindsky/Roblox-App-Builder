# Roblox App Builder

This is a starter kit for creating your own Roblox app. There are versions for Ubuntu (Linux distro) and Windows. Please be sure to read the IMPORTANT.txt file in one of the folders.

## WARNING: THIS GITHUB REPOSITORY IS NOT AFFILIATED WITH ROBLOX CORPORATION









# How this is work?:

The Windows and Linux versions work almost the same way; the only major difference is that since Roblox isn't officially supported on Linux, it uses Sober.

Basically, to make it work on Windows, you need to install Roblox.exe, but for Linux, it uses Sober, as I said.
